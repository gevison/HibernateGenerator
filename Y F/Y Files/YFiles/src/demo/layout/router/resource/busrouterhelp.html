Help

A bus is composed of one or several interconnected hub nodes and the edges that connect the regular nodes to the hubs. For clarity, the hubs and edges of the same bus are displayed in a common color.

Usage

Adding new nodes
Clicking at the desired location in the view creates a new regular node. Hubs are not created manually, this demo creates them automatically where edges meet.
Adding new edges
An edge is created by dragging the mouse from the (unselected) start node to the target node. By holding the Shift key during button press or release, it is possible the start or end an edge at another edge, respectively. Subsequently, colors are updated to reflect the changed bus structure and an automatic routing of the new edge is done.
Removing nodes and edges
Selected nodes and edges can be removed using the DELETE key or the corresponding icon in the tool bar. Stub paths created by a deletion are removed automatically.
Routing buses
The routing algorithm and its properties are available in the tool bar. Use Route All to calculate a new routing for all buses and Route Selected to route each bus in which at least one edge, one hub or one adjacent regular node is selected.
Automatic routing
Moving and resizing of nodes triggers an automatic routing of their connections in which the remainders of the affected buses are kept fixed. New connections and single-edge buses are routed automatically, too.

Settings

Grid
Affects both the editor and the routing algorithm. If enabled, a grid of the given spacing is shown in the view and the algorithm assigns positions on grid coordinates if possible.
Snapping
Specifies whether snap lines are activated.
